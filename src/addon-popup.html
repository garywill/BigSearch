Handily use / switch any one or multiple (uninterruptedly) search engines (or search any websites) via a flexible tool.

  • After finish inputting, click button(s) in the right column of the table to perform an operation (e.g. search) in the corresponding engine (or website/database). A new page will be opened to display the results.

  • The last used button is highlighted, and can be used again by pressing Enter.

  • Double-click an entry in the history to quickly insert the entry text to the input box
    • In single-line edit mode, double-clicking replaces the text
    • In multi-line edit mode, double-clicking inserts to cursor


  • Don't forget to choose a UI style and color nice to your eyes ~ We have both simplicity and fancy UI

  • Keyboard shortcut
    • Open popup. Firefox: Ctrl+Alt+S | Chrome & othersï¼šCtrl+Shift+S
    • Set selected text as search term (then use open popup). Firefox: Ctrl+Alt+D | Chrome & othersï¼šCtrl+Shift+D
