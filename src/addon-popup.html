Handily use / switch any one or multiple (uninterruptedly) search engines (or search any websites) via a flexible tool.

  • After finish inputting, click button(s) in the right column of the table to perform an operation (e.g. search) for the corresponding website (or database). A new page will be opened to display the results.

  • The last used button is highlighted, and it can be called again by pressing Enter (if in the same catagory).

  • Double-click an entry in the history to quickly insert the entry text to the input box. (In single-line edit mode, double-clicking replaces all text; In multi-line edit mode, double-clicking inserts to cursor)
